### Project Requirement

- Python 3.9
- Linux Ubuntu
- Docker, Dokcer-Compose
- PostgreSQL

### Set up the project

- Clone the project from its repository
- Get the .env file and set the respective variables
- Create the database and note the name, port, db user & password
